#h5Chat


## 基于HTML5的实时聊天工具的设计与实现

#### 功能：
1. 一对一聊天
2. 多对多聊天
3. 一对多发送消息
4. 聊天表情发送
5. 图片发送
6. 个人信息修改


#### 运行方式

1.克隆仓库

`git clone https://git.coding.net/vuji/HTML5Chat.git`

2.安装nodejs环境

[http://nodejs.cn](http://nodejs.cn)

3.全局安装gulp

`npm install --global gulp`

4.安装node_modules

`cd HTML5Chat-master && npm install`

6.使用gulp编译sass文件、压缩css、js等文件

`gulp`

7.运行项目

`npm start`
